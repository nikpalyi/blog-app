## How to run this app:
Clone or download this repo:
```
git clone https://github.com/nikpalyi/blog-app.git
```
After cloning or download, run:
```
cd blog-app
```
OR
```
cd blog-app-master
```
Run:
```
sbt compile
```

Run:
```
sbt run
```
